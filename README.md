# 🎮 Connect Four AI — Minimax with Alpha-Beta Pruning

## 📌 Overview

This project implements a playable **Connect Four game** where the AI uses the **Minimax algorithm with Alpha-Beta pruning** to make optimal decisions. The goal is to create an intelligent agent capable of competing against a human player efficiently.

---

## 🧠 Key Concepts

* Minimax Algorithm
* Alpha-Beta Pruning (Optimization)
* Game Tree Search
* Heuristic Evaluation Function

---

## 🚀 Features

* 🎯 Human vs AI gameplay
* ⚡ Optimized decision-making using Alpha-Beta pruning
* 🧩 Custom evaluation function for non-terminal states
* 📊 Visual representation of the game board
* 🔄 Turn-based interactive system

## ⚙️ How It Works

1. The game board is represented as a grid.
2. The AI generates possible moves.
3. Minimax explores future states up to a depth limit.
4. Alpha-Beta pruning reduces unnecessary computations.
5. The evaluation function scores board states.
6. The AI selects the optimal move.
